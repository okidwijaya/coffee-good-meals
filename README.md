## Features

- Add Products
- Edit Products
- Add Promo
- Edit Promo
- Sign Up
- Sign In
- Forgot Password
- Update Profile
- Transactions
- History Management

## Installation

1. Clone this repository <repository https://github.com/okidwijaya/coffee-good-meals>

2. Download all packages

3. Configure .env file 

    REACT_APP_HOST = "https://aws-coffee-good-meals.herokuapp.com"

4. Run npm start
